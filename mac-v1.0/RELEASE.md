# OpenClaw Mac Agent v1.0

## 包含内容

- `openclaw-agent` - Mac 平台二进制文件（1.7MB）
- `mac-profile.json` - Mac 配置文件
- `start-mac.sh` - 启动脚本
- `README-mac.md` - Mac 部署说明
- `skills-turix/` - TuriX 自动化 Skill

## 系统要求

- macOS 10.15+ (Catalina 或更高)
- 网络连接到 VPS QUIC 服务器

## 快速开始

```bash
chmod +x openclaw-agent start-mac.sh
./start-mac.sh
```

详细说明见 README-mac.md
